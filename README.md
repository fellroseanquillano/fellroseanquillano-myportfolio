 📂 PROJECTS

GamotFinder: A Trie-Based Offline Drug Reference for Philippine Health Workers 

GamotFinder is a Java-based, offline Command-Line Interface (CLI) application 
developed to address this gap. The system allows health workers to perform two 
critical lookup operations: (1) searching for medicines by partial drug name through 
prefix-based autocomplete, and (2) searching for medicines by symptom keyword. 
The system is powered by the Trie (Prefix Tree) data structure, which enables fast, 
scalable prefix searching that operates independently of the total database size. 
GamotFinder is designed to run on low-specification hardware commonly available 
in barangay health stations and requires no internet connection, external libraries, 
or third-party dependencies. 

Repository:
https://github.com/fellroseanquillano/GamotFinder
